    # Игра "Угадай число" на Python

    Это простая консольная игра, написанная на Python, где компьютер загадывает случайное число, а игрок должен его угадать за минимальное количество попыток.

    ---

    ## Возможности
    - Компьютер загадывает случайное число в диапозоне от 0 до 100.
    - Программа предлагает выбрать уровень сложности.
    - Игрок вводит свои догадки.
    - Программа подсказывает, было ли загаданное число больше или меньше догадки.
    - Подсчет количества попыток.
    - Возможность сыграть несколько раундов.

    ---

    ## Требования
    - Python 3.8+ (игра использует только стандартную библиотеку Python, никаких дополнительных установок не требуется).

    ---
